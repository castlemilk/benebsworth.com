# Design Context — benebsworth.com

## Target audience
Recruiters, hiring managers, and fellow engineers (platform/SRE/cloud) evaluating Ben Ebsworth; peers arriving from talks, GitHub, or the blog.

## Use cases
- Skim a credible, memorable resume (work history + education) fast.
- Verify certifications at a glance (AWS / GCP / CNCF badges).
- Browse projects and technical writing.
- Be a little delighted — the site is also a playground.

## Brand personality / tone
Technical, precise, a little playful. Confident senior platform engineer. The landing is a generative DFS "snake" word-grid — clever, not corporate. The rest of the site should feel like a well-crafted **technical dossier**: editorial, image-rich, refined — not a generic SaaS template.

## Visual system (established + approved)
- **Dark canonical.** Background near-black tinted (`#0a0a0c` / `#131318`), neutrals tinted toward the brand hues — never pure black/white, never flat gray-on-color.
- **Accents (use as a meaningful coding system, not decoration):** blog teal `#00e0b8`, project purple `#7c5cff`, about orange `#ff7a59`. Map accents to meaning (e.g. provider/company), don't scatter them.
- **Type:** JetBrains Mono for labels/headings/technical metadata; Inter for body/long-form. Strong hierarchy, fluid scale.
- **Motion:** high-impact orchestrated reveals over scattered micro-interactions; scroll-reveal stagger, pointer-tracked spotlight/tilt on cards; exponential ease-out; ALWAYS honor `prefers-reduced-motion`.
- **Avoid AI-slop:** no glassmorphism everywhere, no identical icon+heading+text card grids, no gradient text on headings/metrics, no purposeless neon glow. Vary spacing/rhythm; not everything in a card; left-aligned editorial compositions over centered blandness.

## Constraints
Next.js 16 App Router, **static export** (`output: 'export'`), React 19, Tailwind v4 (CSS `@theme`), shadcn base-nova. Lean/custom animation components (no heavy animation lib). Protobuf is the canonical content data model (`proto/content.proto` → `lib/gen/content.ts`).
